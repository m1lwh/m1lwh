✨ Особенности
🛍️ Витрина товаров - карточки с фото, ценой, кратким описанием, тегами и кнопкой «Купить».

🔍 Поиск и сортировка – фильтрация товаров по названию и описанию, сортировка по цене.

🌐 Переключение языка – русский/английский с сохранением выбора в localStorage.

🌗 Светлая/тёмная тема – переключается кнопкой в шапке.

🎮 3D-наклон карточек – при наведении мыши карточка наклоняется в сторону курсора.

💬 Отзывы – реальные отзывы покупателей с ссылками на профили и заказы FunPay.

❓ FAQ – аккордеон с часто задаваемыми вопросами (двуязычный).

🎁 Бонус за отзыв – модальное окно с вводом секретного ключа (rmhxucdldcqmrpxkljng), после которого открывается bonus.html с гайдами и коллекцией значков Steam.

🖼️ Галерея значков Steam – 25 красивых значков с ссылками на покупку карточек на торговой площадке.

📚 Полезные гайды – на странице бонуса собраны инструкции: оптимизация игр для ПК и Android, киберспортивные разрешения, Twitch-бот для дропов, 300k аватарок и другие.

🔔 Милый звук – при клике на гифку в шапке проигрывается звук «ня».

1. 🚀 Как запустить локально
Клонируйте репозиторий
git clone https://github.com/m1lwh/m1lwh.git
cd m1lwh

2. Откройте index.html в любом браузере (двойным кликом или через Live Server).
Всё работает без сервера и сборки – это статический сайт.

3. Для бонусной страницы – также можно открыть bonus.html напрямую, но она предназначена для перехода после ввода ключа.

➕ Как добавить товар
1. Подготовьте картинку товара (желательно 600×400 px) и поместите её в assets/products/.

2. Откройте index.html, найдите массив productsData в разделе <script>.

3. Скопируйте любой существующий объект товара и вставьте новый, изменив поля:

id – уникальный идентификатор,

name – объект с названиями на русском и английском,

price – цена в рублях,

image – имя файла картинки,

shortDesc – краткое описание (для карточки),

fullDesc – полное описание (для модального окна),

platform – платформа (строка),

type – категория (объект для RU/EN),

stock – наличие (или '∞'),

autoDelivery – true/false,

popular – true, если товар должен отображаться во вкладке «Популярное».

4. Сохраните и залейте изменения на GitHub – сайт обновится автоматически.

🎁 Бонусная система
На главной странице в шапке есть кнопка «🎁 Бонус за отзыв».

При клике открывается модальное окно с полем ввода.

Секретный ключ: Выдаеться после 5 - звездочного отзыва выдаеться ключ

После ввода правильного ключа происходит переход на bonus.html, где:

Благодарственное сообщение.

Галерея из 25 красивых значков Steam с ссылками на покупку карточек.

Полезные гайды и ссылки (оптимизация ПК/Android, киберспортивное разрешение, бот для Twitch-дропов, 300k аватарок и др.).

Блок с услугами продавца (ссылки на FunPay).
